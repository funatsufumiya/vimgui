# imgui for vlang (gg)

***WORK IN PROGRESS***

Currently clang is only supported (`v -cc clang`)

## Usage

```
$ git clone --recursive https://github.com/funatsufumiya/vimgui ~/.vmodules/imgui
$ v run ~/.vmodules/imgui/example/example.v
```
